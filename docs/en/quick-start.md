# Sports Club Management System (SCMS)

## Quick Start Guide

Welcome to the Sports Club Management System. This concise guide will help you get up and running quickly.

### 1. Accessing the System

1. Open your web browser.
2. Navigate to the system URL (e.g., `http://localhost/sports-v2/`).
3. **Public View:** You will see the public directory of registered clubs. No login is required to browse basic club details.
4. **Admin Login:** Click the "Login" button at the top right (or navigate to `/login.php`).

### 2. Logging In (Administrators)

- Enter your registered email address and password.
- Click **Login**.
- Upon successful login, you will be redirected to the **Admin Dashboard** which displays system statistics and recent activity.

### 3. Registering a New Club

1. From the sidebar menu, click on **Register Club**.
2. **Basic Details:** Enter the club name and contact details.
3. **Registration Number:** Click the "Generate" button to automatically create a standardized registration number based on the selected District and Division.
4. **Location Details:** Select the District, Divisional Secretariat, and Grama Niladhari Division.
5. **Officials:** Enter information for the Chairman and Secretary (Name, Phone, Address).
6. **Equipment:** Allocate initial sports equipment to the club.
7. Click **Save/Register**. The system will confirm the registration and redirect to the club details page.

### 4. Updating Club Status (Reorganization)

1. Navigate to **Clubs** > **Reorganizations**.
2. Search for the club you want to update.
3. Add a new reorganization date and details. This maintains a historical record of the club's active status renewals.

### 5. Generating Reports

1. Go to the **Reports** section from the sidebar.
2. Select the type of report (e.g., District Statistics, Equipment, Registered Clubs).
3. Apply filters (date range, specific districts).
4. View the charts or click **Export** to download the data as PDF, Excel, or Print directly.

_For more detailed instructions, please refer to the complete User Manual._
